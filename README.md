# Chinese American Autism Resource Center
# 华裔自闭症资源中心

A bilingual (English/Chinese) resource website for Chinese American families affected by autism.

## Features

- 🌐 **Bilingual**: Full English and Chinese content with language toggle
- 📱 **Responsive**: Works on desktop, tablet, and mobile
- ♿ **Accessible**: Semantic HTML with ARIA labels
- 🚀 **Static**: No backend required, hosts free on GitHub Pages
- ✏️ **Easy to edit**: Content marked with clear editable comment blocks

## Pages

| Page | Description |
|------|-------------|
| Home | Mission, overview, and statistics |
| What is Autism | Definition, early signs, diagnosis, cultural perspectives |
| Resources | Therapies, national organizations, Chinese-speaking resources |
| Navigate the System | Insurance, IEP/504, early intervention |
| Community | Support groups, family stories, tips |
| FAQ | 8 common questions with expandable answers |
| Contact | Contact form and emergency resources |

## Getting Started

1. Open `index.html` in your browser to preview the site
2. Read [TUTORIAL.md](TUTORIAL.md) for instructions on editing and deploying
